Information on Plastic Recycling
In everyday life plastic often seems to be everywhere. This app is meant to give some information about the plastics we use on a daily basis. The Resin Identification Code (RIC) is a symbol that appears on products to indicate the raw material used to manufacture an item. The following classifier allows the identification of different types of plastics. It provides some information about the environ-mental problems associated with the classified resin.

Impressum
This application is based on the open source concept: all data used to create this app is openly accessible to everyone. In this application, only the seven types of plastics classified according to ASTM International standard have been taken into account for classification. It is however conceivable to extend the recognition to other labels and trademarks, and, depending on the data set used to train the algorithm, even the recognition of specific objects and the like. The data set of the present application was created during excursions to shopping centers, hardware stores and in the daily routine. Since the data set was relatively small (~ 500 images), the user may excuse misclassifi-cation. However, you are welcome to participate in the project and take it to the next level - I'm looking forward to hear from you.
